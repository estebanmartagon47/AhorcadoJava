# Juego del Ahorcado - Java

¡Bienvenido al **Juego del Ahorcado**!  
Este es un proyecto en **Java** que simula el clásico juego del Ahorcado, donde el jugador debe adivinar una palabra antes de quedarse sin intentos.

## Características

- Juego interactivo en consola.
- Selección aleatoria de palabras.
- Muestra el progreso de la palabra con guiones bajos.
- Control de intentos fallidos y final del juego.
- Mensajes de victoria o derrota según el resultado.
